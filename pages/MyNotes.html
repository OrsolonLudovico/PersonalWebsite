How to handle a .onepkg file
Why .onepkg?:

The original format allows for a clear distinction between the various sections (often there are a lot of pages). Converting it to PDF is possible but you'll drown in hundreds of unorganized pages; the sections are very important.

Open a .onepkg file:

The easiest way to open a .onepkg file is to have Microsoft OneNote installed on your computer or mobile device. You can download it from onenote.com. Once you have it installed, you can open the file by double-clicking it or by going to File > Open inside the app.

Export to PDF (not recommended):

After opening the file in the OneNote app, you can export it to PDF by going to File > Export > Notebook > PDF format. You can also choose to export individual sections or pages if you don't need the entire notebook.

Alternative methods (not recommended):

If you really don't want to download OneNote (diehard Linux/Mac user, I see you), you can try converting the file to PDF using online services, but in my experience, they usually don't work well. You could also try opening the file from the OneNote web app and printing it to PDF from there, but it's quite clunky and not guaranteed to work.
